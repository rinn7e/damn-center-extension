Adds customizable left and right padding to web pages with support for theme-aware background colors and SVG patterns.

Damn Center (the page!) adds customizable left and right padding to web pages on the fly. Adjust padding width, select side shifting, and align layout grids. Customize base colors, select beautiful SVG background patterns (grids, dots, stripes, carbon fiber, lattice), and automatically sync with system Light/Dark mode.

Includes a vertical split alignment ruler and a smart window maximization detector that suspends padding automatically when the browser window is tiled or resized to optimize screen space.
